## Fellowship tracks

Fellows apply to one of two tracks:

**Sponsored track** — Teams are matched to a company sponsor's defined project. Companies pay a sponsorship fee; fellows on lead-sponsored projects receive a $1K stipend upon completion. Sponsors submit project briefs in advance.

**Indie track** — Teams bring their own idea or work on a faculty-sponsored research project. Access to all mentorship and programming. Eligible for grand prizes. No guaranteed stipend.

<Callout type="warning" title="Team applications">
Each applicant must submit an **individual** application even if applying as a pre-formed team. Applicants who want to be matched together should list teammates' names in their application. They may use the same essay responses.
</Callout>

## Fellowship structure

Fellows commit approximately **10–15 hours per week** over the summer. The program includes:

- Weekly workshops from sponsor and mentor organizations
- Structured mentorship (mentors commit ~2–4 hours/month)
- A dedicated technical help channel for async unblocking
- Regular check-ins with fellowship facilitators (TAs)
- Demo opportunities throughout the cohort
- Anthropic Claude API credits for all teams (managed via MDPlus org on platform.claude.com)

## Anthropic API access

MDPlus has a Claude API organization account set up at [platform.claude.com](https://platform.claude.com) under the org **MDPlus**, seeded with **$30K in credits** from Anthropic (confirmed title sponsor). Arvind Rajan manages the org account.

Each team receives a named API key with an allocated credit budget. Usage per team can be tracked in the console. Individual workspaces per team are also an option for more granular access control.

Contact **Arvind Rajan** to get an API key issued for your team.
