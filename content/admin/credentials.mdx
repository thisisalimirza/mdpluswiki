## Access request process

1. New leader requests access via the onboarding checklist
2. Chief of Staff verifies role and need
3. Relevant admin grants minimum necessary access
4. Access logged in this document

## Offboarding checklist

When a leader departs:
- [ ] Remove from all Slack channels
- [ ] Remove from Google Groups
- [ ] Remove from GitHub org
- [ ] Remove from Vercel team
- [ ] Rotate any shared passwords they had access to
- [ ] Archive their email (if applicable)
